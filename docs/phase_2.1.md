## **通过4个月的K线区间数据，找出具有右侧交易的上涨形态**

---

比如，我们使用 python scripts/quant_picker.py --t_date 2024-03-01 ，这个命令中的 --t_date就是T日，它表示执行的日期
是end日期，不能在往后泄露未来的时间了。

### 一、 这4个月的区间时间是下面的规则分配的

---

第 1 个月区间：
    终点 = T（你指定的日期）
    起点 = 从 T 往前 22 个交易日
    在这 22 天里找到：
        最高价 H1 及其日期 date_H1
        最低价 L1 及其日期 date_L1
第 2 个月的终点 = 第 1 个月里最早出现的那个极值日期
    比较 date_H1 和 date_L1
    谁的日期更小（更早），谁就作为第 2 个月的终点
第 2 个月区间：
    终点 = min(date_H1, date_L1)
    起点 = 从这个终点再往前 22 个交易日
    再找到 H2、L2、date_H2、date_L2
第 3、4 个月 完全重复上面逻辑。

---

### 二、 上述这样做的原因及好处

每一段区间都锚定在真实资金反转的日期上
    不是机械切 22 天，而是按价格行为切
    这样算出来的：
    区间涨幅
    回调幅度
    回抽幅度
    累计涨幅
全部是真实资金走出来的，不是假区间
这就是我期望要的：能计算每个连续区间的累计涨幅，用来识别4 种形态： N 型（最强）、W 底（双底）、H 型（高位横盘，强者恒强）、 V 底（超跌反转，爆发力最大）

### 三、 伪python代码流程

#### 变量说明

T：最终结束日
window = 22：每个区间交易日数
N = 4：要取 4 个区间（4 个月形态）

#### 步骤 1：取第 1 个区间

```
区间1:
end1 = T
start1 = 从 end1 往前推 22 个交易日

在 [start1, end1] 中找到：
high1 = 最高价
low1  = 最低价
date_high1 = 最高价日期
date_low1  = 最低价日期

# 关键：下一个区间的终点 = 更早的那个极值日期
end2 = min(date_high1, date_low1)
 
```

#### 步骤 2：取第 2 个区间

```
区间2:
end2 = min(date_high1, date_low1)
start2 = 从 end2 往前推 22 个交易日

找到：high2, low2, date_high2, date_low2
end3 = min(date_high2, date_low2)

```

#### 步骤 3、4 完全一样递推……

### 四、最终得到 4 个完美区间

得到4组区间：

```
[start1, end1] → high1, low1
[start2, end2] → high2, low2
[start3, end3] → high3, low3
[start4, end4] → high4, low4

```

### 六、 补充四种强势形态的定义

1. N 型（最强，策略的核心）

4 个月高低点表现：

- 先涨
- 再跌（浅调 / 深调都行）
- 再快速拉回
- 最后一个月（最近一月）站在高位

特点： 拉升 → 洗盘 → 吸筹 → 再起，T 日往往就是 第二波起点了，只要来个利好，直接主升浪


2. W 底（双底，趋势反转最稳）

4 个月表现：

- 跌 → 弹 → 再探底（不创新低）→ 再拉起
- 两个低点 几乎持平
- 最后一个月 重心上移

特点： 底部扎实，只要出新闻，直接反转走趋势


3. H 型（高位横盘，强者恒强）

4 个月表现：

- 前面已经涨起来
- 后面 高位横盘 2～3 个月
- 高低点区间很小，不跌
- 最后一月 开始微微抬高

特点： 主力没走，利好一来，直接突破，不回调

4. V 底（超跌反转，爆发力最大）

4 个月表现：

- 前面暴跌
- 最后 1 个月 暴力 V 型反转
- 低点出现在倒数第 2 个月
- T 日正好在 V 型右侧起步

特点： 超跌 + 情绪修复，一个新闻就能连续大阳
