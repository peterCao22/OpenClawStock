# Phase 3: OpenClaw StockMonitorSkill 需求定义 (Draft)

## 1. 项目背景

Phase 2 已经输出了每日的 TOP-20 潜力股列表（包含 N/W/H/V 形态）。
Phase 3 的目标是开发一个“监控技能 (Skill)”，在交易时段对这些股票进行实时/准实时监控，利用 LLM 的能力辅助交易决策。

## 2. 核心业务流程 (待确认)

**初步构想的流程如下：**

1. **初始化**: 每天开盘前 (09:25)，读取 Phase 2 生成的最新 JSON 列表。
2. **行情轮询**: 在交易时间 (09:30-11:30, 13:00-15:00)，每隔 `N` 分钟获取一次 TOP-20 的实时行情。
3. **硬规则触发**: 检查是否满足“异动条件”（如：急涨、大单买入、封板）。
4. **LLM 分析**: 一旦触发异动，调用 LLM：
  - 输入：当前行情、技术形态（来自 Phase 2）、实时新闻（可选）。
  - 输出：分析结论（是主力拉升还是诱多？建议操作？）。
5. **消息推送**: 通过飞书发送告警卡片。
