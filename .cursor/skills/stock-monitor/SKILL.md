# Stock Monitor Skill

This skill allows OpenClaw to monitor a list of top-scored stocks generated by the quantitative screening script and analyze their real-time market data to detect trading opportunities.

## Usage

When the user asks you to start monitoring stocks or check for market anomalies, you should run the following script:

```bash
python scripts/stock_monitor.py
```

### What it does

1. Reads the latest top-20 watchlist from `results/` generated by `scripts/quant_picker.py`.
2. Fetches real-time market quotes using the Moma API.
3. Checks for simple anomalous patterns (e.g., daily price change > 3% and volume ratio > 1.5).
4. When triggered, it invokes the Doubao LLM to perform an expert analysis on whether the stock is breaking out from its "surge-pullback-rebound" pattern.
5. The analysis will result in a clear, short recommendation.

## Extension

In the future, this skill will be extended to automatically push these LLM analyses to Feishu (Lark) so the user can make manual trading confirmations.
